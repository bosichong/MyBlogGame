# 文章访问量速算表

## 一、计算公式

```
最终访问量 = SEO基础 × 质量加成 × (1 + 设计加成 + 段位加成) + RSS加成 + 收藏加成
```

### 各修饰器公式

| 修饰器 | 公式 | 说明 |
|--------|------|------|
| SEO基础 | `(SEO/100) × rand(基础值/3 ~ 基础值)` | 最低保证: max(基础值/10, 1) |
| 质量加成 | `× (1 + 质量/150)` | 50分→×1.33, 100分→×1.67, 200分→×2.33 |
| 设计加成 | `+ SEO基础×设计值×0.003` | 设计100时约+30% |
| 段位加成 | `+ SEO基础×段位²×0.006` | 段位5→+15%, 段位9→+49% |
| RSS加成 | `+ RSS×0.1×(1-天数/15)` | 限新文章14天内 |
| 收藏加成 | `+ 收藏×0.05×(质量/100)` | 限280天内 |

### 文章等级基础值

| 等级 | 基础值 | 最低保证 |
|------|--------|----------|
| Lv1 | 50 | 5 |
| Lv2 | 200 | 20 |
| Lv3 | 500 | 50 |
| Lv4 | 1000 | 100 |
| Lv5 | 1600 | 160 |

## 六、优化方向建议

1. **提升基础访问量**：增加文章等级基础值或SEO权重
2. **平衡加成比例**：当前设计/段位加成相对较低，可适当提升
3. **新增加成机制**：如联盟互助加成、节日活动加成等
