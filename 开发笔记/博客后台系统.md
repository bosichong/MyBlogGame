# 博客后台系统

## 系统定位

**博客后台 = 博客数据管理中心**

提供博客文章、友情链接、评论的查看与管理功能。玩家可以删除不需要的内容，查看博客运营数据。

---

## 系统架构

**独立模块设计，面板列表导航**

采用类似"博客联盟"的导航列表模式，左侧按钮组切换右侧面板。可逐个添加功能面板，不必一次性完成。

---

## 入口位置

游戏主界面右下角按钮组，新增按钮：


## 技术实现

### 场景结构

```
scenes/
└── blog_dashboard/
    ├── blog_dashboard.tscn      # 主场景
    ├── dashboard_panel.tscn     # 数据看板面板
    ├── articles_panel.tscn      # 文章管理面板
    ├── friendlinks_panel.tscn   # 友链管理面板
    └── comments_panel.tscn      # 评论管理面板
```

### 脚本结构

```
scripts/
└── blog_dashboard/
    ├── blog_dashboard.gd        # 主控制器（按钮组切换）
    ├── dashboard_panel.gd       # 数据看板逻辑
    ├── articles_panel.gd        # 文章管理逻辑
    ├── friendlinks_panel.gd     # 友链管理逻辑
    └── comments_panel.gd        # 评论管理逻辑
```

### 导航切换逻辑（参考 lm_mian.gd）

```gdscript
# blog_dashboard.gd

@onready var panels = [
    $bg/面板组/panel1,  # 数据看板
    $bg/面板组/panel2,  # 文章管理
    $bg/面板组/panel3,  # 友链管理
    $bg/面板组/panel4,  # 评论管理
]

@onready var buttons: Array[Button] = [
    $bg/按钮组/vb/b1,
    $bg/按钮组/vb/b2,
    $bg/按钮组/vb/b3,
    $bg/按钮组/vb/b4,
]

func show_panel(index: int) -> void:
    for i in range(panels.size()):
        panels[i].visible = (i == index)
    set_button_pressed(index)

func set_button_pressed(index: int):
    for i in range(buttons.size()):
        buttons[i].set_pressed(i == index)
```
