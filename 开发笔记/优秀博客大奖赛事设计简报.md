# 优秀博客大奖赛事设计（v2 — 等级驱动版）

## 判定规则

| 玩家等级 | 获奖结果 | 称号 |
|---------|---------|------|
| `lv < 20` | 无奖项 | 无 |
| `20 ≤ lv < 50` | 提名 | 博客新星 |
| `50 ≤ lv < 80` | 入围 | 博客新锐 |
| `80 ≤ lv ≤ 100` | 优秀博客大奖 | 博客之星 |
| 连续同结果 | 蝉联触发 | 追加蝉联提示 |

## 蝉联判定

- 每次揭晓时记录本次结果等级（none/nominate/shortlist/win），与上一次对比，相同且非 NONE 则触发蝉联
- 蝉联提示：
  - 提名蝉联：再次获得提名，继续精进，未来可期！
  - 入围蝉联：再次入围，实力稳健，距大奖仅一步之遥！
  - 获奖蝉联：蝉联优秀博客大奖！传奇仍在继续！

## 揭晓文案模板

### 无奖项（lv < 20）

> 继续努力提升博客质量，下一届期待您的参与！
>
> 评语：「坚持写作，时间会给你答案。」

> 您获得称号「博客新星（年份）」！
> [蝉联]再次获得提名，继续精进，未来可期！
>
> 评语：「每一篇认真的文字都值得被看见。」

> 您获得称号「博客新锐（年份）」！
> [蝉联]再次入围，实力稳健，距大奖仅一步之遥！
>
> 评语：「您的博客代表了中文独立博客的中坚力量。」

### 获奖（80 ≤ lv ≤ 100）

> 第 N 届「中文优秀博客大奖」获奖名单正式公布——
> 本届共有 6 位博主荣获优秀博客大奖，「博客名」是其中之一！

>
> 评语：「您的博客是中文独立博客的标杆与灯塔。」

## 三阶段文案规则

- **通知阶段**：只说大赛启动邀请，不提"已提名/已入围/已蝉联/传奇人物"等预判
- **评选阶段**：只说评审进行中，不提玩家所处状态
- **揭晓阶段**：按等级动态生成（上表模板）

## 场景剧本

### 通知阶段

> 卧室白天 → 玩家坐电脑前 / 狗发呆 → 邮件提示 → 切邮箱界面 → 点击展开正文 → 退出返回

### 评选阶段

> 纯黑底字幕逐条淡入 → 显示完毕自动返回

### 揭晓阶段

- **获奖（lv≥80）**：现场颁奖舞台 → 大屏幕滚名单 → 玩家名高亮 → 领奖定格 → 显示评语
- **提名/入围/无奖项（lv<80）**：卧室邮件通知 → 同通知阶段流程

## 数据流

```
通知时间到 → 显示中立邀请弹窗 → 确认 → 邮箱动画（无预判）
评选时间到 → 显示中立评选弹窗 → 确认 → 字幕叙事
揭晓时间到 → 显示结果弹窗 → 确认
                ↓
         读取 Blogger.level
                ↓
         判定等级区间 → 结果等级
                ↓
         对比 award_history → 蝉联
                ↓
         动态生成文案 → 现场/邮件场景
                ↓
         记录 award_history + 标记里程碑
```
