# 第一章：博客启蒙期（2001-2005）

**主题**：博客起步，建立基础

**时间跨度**：3年（游戏内时间）

---

## 0. 时间系统

### 0.1 游戏起始时间

| 字段 | 值 | 含义 |
|------|------|------|
| year | 2001 | 游戏开始年份 |
| month | 12 | 12月 |
| week | 4 | 第4周 |
| day | 7 | 周日 |

**代码位置**：`TimeData.gd`
```gdscript
var current_year: int = GAME_START_YEAR  # 2001
var current_month: int = 12
var current_week: int = 4
var current_day: int = 7
```

### 0.2 域名/主机到期时间

| 项目 | 起始时间 | 到期时间 |
|------|----------|----------|
| 域名 | 2001-12-4-7 | 2002-12-4-7 |
| 主机 | 2001-12-4-7 | 2002-12-4-7 |

**代码位置**：`yun.gd`

### 0.3 时间暂停机制

游戏开始时，`time_stop = true`，游戏时间暂停运行。

发布第一篇博文后，`time_stop = false`，游戏时间恢复正常。

### 任务3：第一篇博文（玩家主动）

**任务ID**: task_003_first_article

**剧情文案**:
> "写下了《你好，世界》"
> "这是我第一次在自己的网站上发布内容。"
> "没有论坛的嘈杂，没有版主的删帖，只有我自己。"

**触发方式**: 玩家主动点击"写文章"

**前置条件**: 博客已上线

**时间条件**: `2001-12-4-1`（周一）

**文章类型**: 第一篇博文（1级）

**体力消耗**: 20

**操作流程**:
1. 玩家点击"写文章"按钮
2. 选择文章类型"第一篇博文"
3. 输入标题"你好，世界"
4. 可选：输入简短内容
5. 点击"发布"

**完成后文案**:
> "发布成功！"
> "这是我的第一篇博文。"

**奖励**:
- 经验值 +10
- 文学技能 +2
- 编程技能 +1
- 访问量 +15
- 成就解锁："第一篇博文"
- 称号获得："博客新人"

**后续任务**: task_004_baidu_indexed

**解锁事件**:
```
发布第一篇博文后触发 post_event 事件
任务ID: unlock_blogging_start
条件: first_post_eq_1

执行动作:
1. UNLOCK_INITIAL_TASKS → 解锁16个日常任务
2. START_GAME_TIME → 启动游戏时间 (time_stop = false)
3. SHOW_NOTIFICATION → 显示解锁提示
```

**代码钩子**:
```
on_task_complete("task_003_first_article", func():
    add_exp(10)
    add_skill("literature", 2)
    add_skill("programming", 1)
    add_visitor(15)
    unlock_achievement("first_article")
    unlock_title("博客新人")
    # 触发解锁事件
    trigger_event("unlock_blogging_start")
)
```
